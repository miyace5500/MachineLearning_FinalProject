# MachineLearning_FinalProject

## 專案簡介
本專案使用 Kaggle MOOC 資料集，分析線上學習平台學習者的行為，並預測學習者是否可能退課。專案包含以下部分：
1. 資料前處理（Data Preprocessing）
2. 探索性資料分析（Exploratory Data Analysis, EDA）
3. 教師式學習問題：退課預測（Logistic Regression, Decision Tree）
4. 非教師式學習問題：學習者分群（KMeans Clustering）
5. 視覺化與結果觀察（可視化圖表與模型評估）

## 資料來源
Kaggle MOOC Final Dataset  
[資料集連結](https://www.kaggle.com/datasets/samyakjhaveri/mooc-final/data)
